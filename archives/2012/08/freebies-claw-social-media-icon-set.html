Freebies: CLAW Social Media Icon Set

7 August 2012 Comments

Belakangan ini saya senang sekali belajar menggunakan tool vektor di Adobe Photoshop. Dari mulai menggambar objek sederhana, bermain-main dengan shape dan path tool, sampai tracing objek menggunakan pen tool. Saya ingin berbagi salah satu hasil latihan saya, Social media icon set yang saya beri nama “CLAW Social Media Icons”.

Icon Set ini terdiri dari 16 ikon sosial media dengan 2 variasi warna, hitam putih dan berwarna, dengan 2 variasi ukuran 24px dan 48px dalam format PNG transparan. Selain itu saya sertakan juga file PSD dengan masing-masing ikon dalam bentuk layer vector shape sehingga dapat dimodifikasi warna dan ukurannya sesuai keperluan.

Semoga bermanfaat
